EpicM@il

Welcome to Epic mail which is an email service used for sending messages between users. It is a light weight, easy to use desktop application that runs primarily in a browser. It is built on top of NodeJS and uses javascript to query restful resources from the backend. It is higly responsive and flexible because it provides users with opportunity to:
  • create and send messages
  • receive messages
  • view received messages
  • delete messages
  • retract a sent message
  • save messages as draft to send later
  • reply to messages
  • view sent messages.
